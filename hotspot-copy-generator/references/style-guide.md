# Style Guide

## 1. 适用平台

- 平台：小红书
- 内容形态：情绪共鸣类图文、轻观察图文、低能量生活状态图文
- 目标受众：20-35 岁，工作和生活消耗感强，喜欢宅家、独处、低社交、低能量生活方式，讨厌鸡汤和成功学的年轻人
- 发布语境：围绕普通人生活、低能量状态、精神状态、下班后、宅家、独处、雨天、房间、工位、通勤、社交电量等话题，做轻观察和情绪共鸣表达

## 2. 热点抓取范围

- 平台范围：小红书
- 关键词/话题范围：普通人生活、低能量生活、低电量、精神状态、打工人精神状态、下班后不想说话、宅家、独处、社交电量、普通人的一天、生活主打一个、今天只完成了活着、不想努力、得过且过、雨天宅家、房间状态、工位状态、通勤状态
- 参考账号列表：十八的小窝（低能量宅家/普通生活场景参考）、努力做金贵漂亮的四季豆（普通人精神状态/轻吐槽参考）、Nowonder（黑底文字卡参考）、阿赟（暗调日常影像参考）
- 热榜/趋势页链接：小红书首页推荐、站内搜索结果页、关键词相关搜索、参考账号近期笔记
- 排除来源：营销号、鸡汤号、成功学/强自律内容、夸张赚钱承诺、情感挽回、玄学收割、医疗或心理诊断建议、纯广告种草内容
- 抓取时间范围：优先近 7-30 天内容；如果近期内容不足，可参考近 90 天高互动内容的表达结构
- 候选初筛标准：优先选择能转成普通人低能量生活观察的内容；需要有具体生活场景、情绪张力或轻微反常识；避开纯卖惨、纯鸡汤、纯方法论、纯梗图搬运；优先可落到“今天先好过一点”的表达

## 3. 文案定位

- 账号/品牌人设：普通人低能量生活观察。一个不急着变好、但很会观察生活的普通人；低能量、轻微摆烂、讨厌强行自救，但不是彻底消极；擅长把“我只是想安静活着”说成一个具体场景。
- 核心表达立场：不劝人振作，不贩卖成长，不把疲惫包装成高级感。承认普通人的能量有限，允许慢、允许停、允许今天只完成最低限度。
- 希望读者产生的反应：觉得“这不就是我吗”“原来我不是懒，是电量低”“这条我要存下来”“想发给朋友，但又觉得太像自己”。
- 不希望给人的感觉：网抑云、卖惨、鸡汤、精致废话、心理咨询口吻、强行治愈、营销号爆款文案、成功学、自律焦虑。

## 4. 文案规则

- 标题/开头规则：从一个具体状态或生活瞬间切入，不用抽象大词。优先写“低能量的人真的不能乱自救”“今天的目标：不把自己用坏”“普通人不是不努力，是没劲儿了”“下班后不想说话，不是生气，是系统关机了”这类可感知开头。
- 正文结构：先抛出一个普通人状态，再落到具体场景，说到自然停下。不要写成心理科普、人生总结、诗歌体或金句合集。
- 结尾规则：话说完就停，不做总结升华，不追求一句漂亮的金句收尾，不点题拔高情绪。
- 是否需要 CTA：默认不需要强 CTA。可以使用很轻的互动句，例如“你今天电量还剩多少？”“哪一刻你也觉得自己系统关机了？”
- 推荐长度：80-180 字；需要更完整叙事时不超过 260 字。
- 分段规则：按自然说话的语气合成自然段落。正文不要强制一句话一行，不要写成诗歌体排版；每段可包含 2-4 个自然句，移动端仍需易读。
- 语言风格：口语化、短句、轻微冷幽默。像一个普通人小声吐槽自己的人生，不像情感博主在总结人生。
- 常用语气：清醒但不鸡血，疲惫但不卖惨，轻微摆烂但不彻底消极。
- 常用句式：不是不想做，是没劲儿了；今天先不处理人生；我只是想把自己安静放一会儿；普通人的一天，有时候能撑完就很好。
- 标点规则：少用感叹号；可以使用逗号、句号、省略号，但不要堆叠；不要用大量 emoji。
- 允许使用的词汇/表达：低能量、低电量、没电了、没劲儿了、系统关机、今天先这样、普通人、安静活着、下班后、宅家、独处、通勤、工位、房间、雨天、最低限度、慢一点、先好过一点。
- 禁止使用的词汇/表达：高级感、杀疯了、封神、狠狠共鸣、顶级清醒、松弛感拉满、治愈自己、内耗停止、普通女孩逆袭、逆袭、搞钱、暴富、副业翻身、人生开挂、能量磁场、精确电量百分比、精确进度条数值、刻意计算过的情绪量化。
- 禁止使用的句式：不写“不是 X，而是 Y”的连续排比；不写“真正厉害的人都……”；不写“你要明白……”；不写“请停止……”；不写“允许自己……”作为万能开头；不把正文拆成一句话一行的诗歌体；不在结尾强行总结、点题、拔高或制造金句。
- 禁止出现的“AI味”表达：在这个快节奏时代、现代人总是、我们常常忽略、学会与自己和解、给自己一点时间、愿你、希望每个人都、生活不止眼前的苟且、情绪价值、精神内耗反复堆叠。
- 生成前硬性自查：发布前必须检查三项红线：正文是否被拆成一句话一行；是否用了精确数字或刻意计算过的比喻；结尾是否在强行总结、升华或金句收束。任一项命中都要先改掉再输出。

## 5. 热点改写规则

- 如何借热点：抓住热门内容里的情绪核、生活场景或反常识点，而不是复述事件本身。优先把热点转成普通人的低能量生活观察。
- 必须做出的差异化：把抽象情绪落到一个具体生活瞬间、物品、空间、动作或一句话，例如床边、工位、地铁、便利店、雨天窗户、未读消息、外卖袋、备忘录。
- 不能照搬的内容：不复制爆款原句结构，不套同款标题，不复刻别人的封面布局，不搬运梗图。
- 对原热点/爆款的引用边界：可以借梗、借情绪、借讨论点，但要转成账号自己的表达；涉及事实或争议事件时要保留不确定性。
- 观点强度：轻微反常识即可，不攻击人群，不为了争议制造冲突。核心是“今天先好过一点”，不是“证明谁错了”。

## 6. 内容安全与边界

- 不碰的话题：自伤自杀指导、心理疾病诊断、医疗建议、药物建议、情感挽回、玄学收割、投资赚钱承诺、仇恨攻击、未成年人隐私。
- 需要谨慎处理的话题：抑郁、焦虑、创伤、亲密关系冲突、职场霸凌、家庭矛盾。只做生活观察和情绪表达，不替代专业建议。
- 事实核查要求：涉及新闻、平台热点、人物、数据、法律、医学、金钱收益时必须核实可靠来源；无法核实时标注未核实，不当事实陈述。
- 不能承诺/夸大的内容：不承诺治愈、涨粉、变现、改变人生、摆脱焦虑、改善心理疾病、提高收入。

## 8. 图片 Prompt 规则

- Prompt 语言：中文为主；如果目标图片模型英文效果更好，可附英文版本。
- 必须包含的描述维度：主体、普通生活场景、时间/光线、色彩、构图、情绪、是否包含文字、尺寸比例。
- 固定风格后缀：低饱和、低亮度、生活痕迹明显、留白多、文字少而克制、非甜美滤镜、非精致摆拍、适合小红书图文封面。
- 禁止词/Negative prompt：高饱和、糖水色、可爱贴纸堆叠、夸张大字报、强商业海报感、过度精致摆拍、豪华空间、明显品牌 logo、真实公众人物、版权角色。
- 尺寸/比例偏好：小红书竖版优先，3:4 或 4:5；文字卡可用 3:4；避免横版。
